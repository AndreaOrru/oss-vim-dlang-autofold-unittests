## Installation

### Using vim-plug

+ Add `Plug 'AndreaOrru/vim-dlang-autofold-unittest', {'for': 'd'}` to your `~/.vimrc`.
+ Restart vim and `:PlugInstall` to install.

## Usage

+ `za` to toggle the fold under the cursor.
+ `zR` to open all folds.
+ `zM` to close all folds.
